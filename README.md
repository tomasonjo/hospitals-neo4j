### Host locally:
[NOTE]

Need to whitelist source in neo4j.conf


`browser.remote_content_hostname_whitelist=http://localhost:8001/`

Start server:
`python http-server`

Get tutorial in Neo4j Browser

`:play http://localhost:8001/hospital.html`
## MapBox:
